# Retail Sales Analytics Dashboard

## Overview

End-to-end Retail Sales Analytics project built using Python, SQL, AWS RDS MySQL, and Power BI.

## Tech Stack

* Python
* Pandas
* Matplotlib
* MySQL (AWS RDS)
* Power BI
* Git & GitHub

## Features

* Generated retail sales data using Python
* Loaded data into AWS RDS MySQL
* Performed SQL-based business analysis
* Conducted data analysis using Pandas
* Created visualizations using Matplotlib
* Built an interactive Power BI dashboard

## Key Metrics

* Total Revenue
* Total Orders
* Total Customers
* Average Order Value

## Dashboard Insights

* Monthly Revenue Trend
* Revenue by Category
* Revenue by State
* Top 10 Products
* Interactive Filters
